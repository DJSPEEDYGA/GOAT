## 🚀 DEPLOYMENT INSTRUCTIONS

### Option 1: Portable Version (Quick Start)

**Steps:**
1. Download `goat-royalty-portable-2.0.0.zip` (17MB)
2. Extract to any folder
3. Open `START-GOAT-APP.html` in any browser
4. **Done!** Application runs immediately

**Best For:** Quick testing, multiple computers, USB drives

### Option 2: Linux Installation (.deb)

**Steps:**
```bash
sudo dpkg -i goat-royalty.deb
sudo apt-get install -f  # Fix dependencies if needed
goat-royalty  # Run application
```

**Best For:** Ubuntu/Debian systems, production servers

### Option 3: Server Deployment

**Steps:**
```bash
# Upload to servers
scp server-package-2.0.0.tar.gz user@93.127.214.171:/tmp/
scp server-package-2.0.0.tar.gz user@72.61.193.184:/tmp/

# Deploy on each server
ssh user@93.127.214.171
cd /tmp && tar -xzf server-package-2.0.0.tar.gz
cd server-package
bash deploy-production.sh
npm start
```

**Target Servers:**
- 🌐 Server 1: 93.127.214.171
- 🌐 Server 2: 72.61.193.184

### Option 4: 8TB Drive Backup

**Steps:**
```bash
# Mount your 8TB drive
sudo mount /dev/sdX1 /mnt/8tb-drive

bash /workspace/nextjs-commerce/backup-to-8tb-drive.sh
```
