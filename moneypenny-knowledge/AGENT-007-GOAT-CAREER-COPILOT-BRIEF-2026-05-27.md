# AGENT-007 GOAT Career Co-Pilot Brief

Classification: owner-local sanitized brief

## Mission

The GOAT Career Co-Pilot lane turns artist, catalog, audience, content, release, and money questions into structured music-business reports. It is inspired by the owner-provided React/Gemini dashboard pattern, but the GOAT version is local-first and does not put API keys in public browser code.

## Best Features To Keep

- Market analysis with SWOT, competitors, trends, and target platforms.
- Audience insights with demographics, psychographics, social behavior, countries, interests, values, and language.
- Content strategy with content pillars, platform ideas, impact ranking, hook frames, and posting schedule.
- Financial plan with revenue streams, release budget allocation, goals, and risk notes.
- Brand kit with statement, palettes, typography notes, logo prompts, and visual direction.
- Press release, sync pitch, fan mail analysis, deal memo review, tour plan, and contacts as future-ready modules.
- Export actions for JSON, CSV, print/PDF, and deck-ready summaries.

## Safety Rules

- Do not hard-code API keys in browser/client code.
- Optional Gemini/OpenAI/other providers must be configured through owner-approved server settings or local environment variables.
- Do not claim live audience, competitor, or market facts unless verified with current sources or real owner data.
- Do not expose private Drive links, client data, unreleased titles, vault phrases, credentials, payment details, or API keys in exports.
- Fresh owner approval is required before sending, uploading, publishing, spending money, or changing DSP/payment settings.
