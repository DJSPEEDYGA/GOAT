AGENT-007 Studio
============

AGENT-007 Studio is the local Codex-style build mode for AGENT-007.

Start here:

1. Run the top-level launcher:
   Launch AGENT-007 Studio.command

2. In AGENT-007:
   - Set Skill to Studio.
   - Open Bridge.
   - Enable Tool Mode when you want AGENT-007 to inspect or edit the approved workspace.
   - Enable Computer Control only when you want owner-approved local computer actions.

3. Check status:
   http://localhost:3333/api/studio/status

What Studio adds:

- A stronger local-builder system prompt.
- A Codex Guardian crew role for disciplined local engineering and AGENT-007 identity protection.
- A status endpoint for launchers and diagnostics.
- A written build contract for the full local-first ChatGPT/Codex-style AGENT-007 path.
- A Business/Office lane for legit Word, Excel, PowerPoint, PDF, Google Drive export, DSP, royalty, and client operations.
- A GOAT Production lane for audio engineering, Pro Tools/AAX planning, film, scoring, voiceover, fingerprinting, release packaging, IP intake prep, and marketing workflows.
- A clean place for future Studio tools, installers, and transfer packs.

What Studio does not fake:

- It does not unlock paid ChatGPT web access inside a local app without official API access.
- It does not run cloud tools without an API key or supported sign-in flow.
- It does not remove owner approval from tool/computer actions.
