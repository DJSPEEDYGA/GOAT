## Why prompts and memory looked like they disappeared

AGENT-007 has several separate layers:

- System Prompt: the current chat instruction field.
- Global System Prompt: what new chats inherit when Set Global is used.
- Project Memory: the saved living memory field.
- Capability Mode: extra mode text for Crew, Icon Art, Audio, and other lanes.
- Loaded profiles: Money Penny, Lexi, Vanessa, Nexus, Sir Codex, Drive Vault, AGENT-007 Protocol, and AGENT-007 Diary can append into memory.

If training text is pasted into the wrong field, changed in a chat but not saved, or overwritten by Set Global/Clear Memory/new chat flow, it can look like AGENT-007 forgot. It usually means the material was not saved into the correct layer.

## Stable training flow

1. Put raw training notes into the correct crew home under `BackupVault/GOAT-Crew-Homes`.
2. Summarize only the durable lesson into AGENT-007 Project Memory.
3. Press Save Memory after updating Project Memory.
4. Keep the base System Prompt short and stable.
5. Store large research, Grammy-winning mix notes, Pro Tools notes, Unreal notes, and model notes as vault files, then save a pointer in memory.
6. Back up `Shared/chat_data/settings.json` before major prompt or memory edits.
7. Do not merge every crew identity into AGENT-007. AGENT-007 coordinates; each crew member keeps their own home.
